What is the structure of your dataset?¶

The dataset contains 183,215 rows and 16 columns, where each row represents a bike trip. It includes temporal, spatial, categorical, and numerical variables related to trip duration, station locations, and user information, The dataset variable are:

duration_min: Duration of the trip in minutes

start_time: Start timestamp of the trip
end_time: End timestamp of the trip
start_station_id: ID of the starting station
end_station_id: ID of the ending station
start_station_name: Name of the starting station
end_station_name: Name of the ending station
start_station_latitude: Latitude of the starting station
end_station_latitude: Latitude of the ending station
start_station_longitude: Longitude of the starting station
end_station_longitude: Longitude of the ending station
bike_id: Unique ID of the bike
user_type: Type of user: Customer or Subscriber
member_birth_year: Birth year of the user
member_gender: Gender of the user
age: Age of the user

What is the main feature of interest in your dataset?¶

The main feature of interest is trip duration duration_min, I will analyze how user age, gender , and type affect trip duration and usage patterns

What features in the dataset do you think will help support your investigation into your feature of interest?¶

The features that will help support my investigation into trip duration are:
age: to estimate user age and explore its relation to trip duration
member_gender: to analyze differences in usage between genders
user_type: to compare ride duration between subscribers and customers

7.Conclusions¶

Summary of Main Findings

  • Trip duration showed some outliers (over 300 minutes) which were removed to clean the data
  • Age data had unrealistic values (over 100 years) which were excluded to improve quality
  • People aged 30–50 tend to have the longest trip durations compared to other ages
  • Customers generally take longer trips than subscribers
  • Females tend to have slightly longer trip durations than males across all user types and days
  • The longest trips occur on weekends (Saturday and Sunday) for both genders
  • Trip durations are generally shorter on weekdays
  • The under 20 age group is not present in the data
